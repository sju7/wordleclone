# Wordleclone

Clone of worlde as a commandline app.
set -show flag to see the chosen word.
Uses https://dictionaryapi.dev/ api to check if guess words are valid.
English words only. 

## Installation
* Install go.
* git pull;
* cd wordleclone;

```bash
go run main.go -show <true/false>
```
