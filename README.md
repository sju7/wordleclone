# Wordleclone

Clone of worlde as a commandline app.
set -word flag to set word for solution.
Uses https://dictionaryapi.dev/ api to check if guess words are valid.
English words only. 

## Installation
* Install go.
* git pull;
* cd wordleclone;

```bash
go run main.go -word <your-word>
```
